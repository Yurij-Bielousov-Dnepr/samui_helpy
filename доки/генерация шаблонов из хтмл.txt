В Django есть инструмент под названием django-adapted (ранее назывался django-html-to-templates), который позволяет автоматически сгенерировать шаблоны Django из существующих статических HTML-файлов.

Чтобы использовать django-adapted, вам нужно установить его, используя pip. Выполните следующую команду в терминале:

Copy code
pip install django-adapted
После установки, перейдите в директорию, содержащую ваши статические HTML-файлы, и выполните следующую команду в терминале:

Copy code
django-adapted
Эта команда создаст каталог templates/, в котором будут содержаться все ваши сгенерированные шаблоны Django.

Обратите внимание, что результаты автоматической генерации могут не всегда быть идеальными, поэтому вам, вероятно, придется редактировать сгенерированные шаблоны вручную, чтобы они соответствовали вашим потребностям.
